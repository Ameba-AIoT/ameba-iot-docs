Introduction

This chapter introduces the default memory layout of RTL8726EA and how to modify the memory layout if needed.

RAM Layout

In total, there are 512KB or 768KB (depending on different series) SRAM on chip, and the size of PSRAM can be 0MB/4MB/8MB/16MB…, which is decided by users. The default RAM layout is illustrated below.

Note

Refer to Section Series Comparison for more information about the SRAM size of different series.
