.. _8721dx_series:

Realtek RTL8721Dx系列产品，为双频Wi-Fi 4 (2.4GHz + 5GHz)和BLE 5.0的双模SoC芯片，具有以下特性：

**系统和平台**

- 多核：

  - Real-M300：

    * Arm Cortex-M55指令集兼容
    * 最大频率345MHz
    * 16KB 4路指令缓存，16KB 4路数据缓存

  - Real-M200：

    * Arm Cortex-M23指令集兼容
    * 最大频率115MHz
    * 16KB 4路指令缓存，16KB 4路数据缓存

- 存储:

  * 512KB on-chip SRAM
  * 内部封装DQ8 PSRAM，最大频率200MHz
  * 支持QSPI Flash，最大频率104MHz

**安全机制**

- 安全启动
- 支持Arm TrustZone-M
- 真随机数生成器（TRNG）
- 硬件加密引擎，支持AES和SHA算法
- 整体或部分Flash加密
- 安全固件独立加密保护（RDP）
- JTAG/SWD调试端口密码保护
- 2KB OTP，其中2Kb可供用户使用

**RF特性**

- Wi-Fi

  * 802.11 a/b/g/n 1 x 1，2.4GHz + 5GHz
  * 支持MCS0 – MCS7 40MHz带宽
  * 省电机制
  * 支持天线分集
  * 低功耗：

    - WoWLAN模式：

      * DTIM=10: 84µA (3.3V, 2.4G)
      * DTIM=10: 74µA (3.3V, 5G)

    - Deep-sleep模式：

  * 接收灵敏度：

    * 12.2µA (3.3V, 25°C)
    * 10.4µA (1.8V, 25°C)

- BT

  * 支持BLE 5.0
  * 支持蓝牙长距离
  * 支持散射网中的piconets（最多8个主节点和3 个从节点）
  * 发射功率：-10dBm ~ 10dBm，典型值为8dBm
  * 接收灵敏度：-99dBm@LE1M

**外设接口**

- 灵活的GPIO配置设计
- 多种通信接口：SPI x 2, UART x 5, I2C x 2
- 硬件Key-Scan接口，最大支持64个按键
- 硬件IR收发器，易于适配各种IR协议
- 支持RTC定时器和10路独立定时器
- 支持8通道PWM定时器和捕获定时器
- 支持7通道通用12位ADC和1个电源量测专用通道
- 支持4通道触摸按键
- 支持8独立通道GDMA
- 支持USB 2.0全速设备模式
- 支持SDIO设备1位和4位模式
- 内置串行LED控制器，用于控制外部LED灯
- 集成像素处理引擎，用于快速处理像素数据
- 集成OSPI显示接口，支持OSPI、QSPI、SPI接口屏幕
- 集成Audio Codec，包含2个数字麦克风通道
- 集成2个I2S接口，最大采样率达384kHz

**封装**

- QFN48封装, 6mm x 6mm
- QFN68封装, 8mm x 8mm
- BGA100封装, 5.1mm x 5.1mm
- 工作电压：

  * 常压：2.97V ~ 3.63V
  * 宽压：1.71V ~ 3.63V

- 工作温度：

  * 常温：-20°C ~ 85°C
  * 宽温：-40°C ~ 105°C

**应用**

凭借优异的超低功耗特性​​、增强的加密策略（PSA Level 2）以及丰富的外设资源，该系列芯片已广泛应用以下产品中：

- 智能家电
- 线控器
- 智能门锁
- 电池摄像头
- 智能遥控器
- Wi-Fi 音箱
- Wi-Fi FullMAC NIC
- BLE网关
- 智能POS机
