  • 芯片使能（CHIP_EN）
  • 概述

概述

芯片使能（CHIP_EN）引脚是一个外部引脚，用于控制系统级复位状态，支持电平复位（默认）、中断复位、脉冲复位三种模式。该引脚在任何模式下均具备强制复位能力。

工作模式

CHIP_EN 有三种工作模式，适用于不同的应用场景。

电平复位模式

电平复位模式是CHIP_EN的默认模式。在这种模式下，当CHIP_EN引脚的低电平持续时间超过消抖时间（100μs~16ms，用户可调）时，将直接触发芯片复位，并在该引脚电平由低变高时重启SoC。

用户只需按下并松开 CHIP_EN 按钮，即可对整个芯片进行复位。

中断复位模式

中断复位模式是CHIP_EN的一种工作模式，主要用于省电场景。在此模式下，当CHIP_EN引脚的低电平持续时间超过消抖时间与短按时间之和时，系统将触发中断而非直接复位。软件可以通过处理该中断来决定是否执行系统复位。

CHIP_EN 按钮的功能类似于智能手机的电源按钮：

  • 短按：触发系统进入低功耗模式或唤醒系统

  • 长按：触发中断，提醒用户是否需要复位系统。

用户选择重启系统，或者系统崩溃，才会进行重启操作。

脉冲复位模式

在脉冲复位模式下，当CHIP_EN引脚的低电平持续时间超过消抖时间时，会直接触发复位操作，SoC将立即重启。

用户可以在启动代码中读取 CHIP_EN引脚的状态，以区分短按和长按操作。

  • 短按：触发使系统冷启动

  • 长按：恢复出厂设置

这种模式通常在路由器中较为常见。

一旦芯片被配置为脉冲复位模式，软件就无法再更改该模式。只有断电才能使芯片回到电平复位模式。

工作模式选择

  • 对于不需要对CHIP_EN引脚进行任何控制的设备，将此引脚连接到高电平。

  • 对于仅需要简单复位功能的场景，使用默认的电平复位模式。

  • 如果既需要复位功能，又需要恢复出厂设置的功能，则使用脉冲复位模式。

  • 对于低功耗场景，且需要在单个引脚上实现电源控制和复位功能时，使用中断复位模式。

API 参考

CHIPEN_WorkMode

Items

Description

Introduction

Configure CHIP_EN work mode

CHIP_EN works in HW reset mode by default.

Parameters

mode: new work mode of CHIPEN, which can be - CHIPEN_HW_RESET_MODE - CHIPEN_INT_RESET_MODE - CHIPEN_PULSE_RESET_MODE

Return

None

CHIPEN_DebounceSet

Items

Description

Introduction

Set the CHIP_EN debounce Time, which works in all work mode

Parameters

debounce: new debounce counter, which can be

  • CHIPEN_DBC_0US

  • CHIPEN_DBC_100US

  • CHIPEN_DBC_500US

  • CHIPEN_DBC_1MS

  • CHIPEN_DBC_2MS

  • CHIPEN_DBC_4MS

  • CHIPEN_DBC_8MS

  • CHIPEN_DBC_16MS

Return

None

CHIPEN_ThresHoldSet

Items

Description

Introduction

Set long press and short press threshold, which only works in interrupt reset mode

Parameters

  • Thres_LP: long press threshold

    • CHIPEN_LP_1S

    • CHIPEN_LP_1P5S

    • CHIPEN_LP_2S

    • CHIPEN_LP_2P5S

    • CHIPEN_LP_3S

    • CHIPEN_LP_3P5S

    • CHIPEN_LP_4S

    • CHIPEN_LP_4P5S

  • Thres_SP: short press threshold

    • CHIPEN_SP_50MS

    • CHIPEN_SP_100MS

    • CHIPEN_SP_150MS

    • CHIPEN_SP_200MS

    • CHIPEN_SP_250MS

    • CHIPEN_SP_300MS

    • CHIPEN_SP_350MS

Return

None

CHIPEN_AckTimeSet

Items

Description

Introduction

Set the ACK threshold for long press interrupt.

If long press interrupt can’t be cleared within Tack, the system will reboot

Parameters

Tack:

  • CHIPEN_ACK_50MS

  • CHIPEN_ACK_100MS

  • CHIPEN_ACK_200MS

  • CHIPEN_ACK_400MS

Return

None
