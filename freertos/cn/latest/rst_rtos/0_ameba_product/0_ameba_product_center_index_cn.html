主要特点

  • Wi-Fi 4 2.4GHz/5GHz双频 + Bluetooth 5.0 LE

  • PSA Level-2 安全认证

  • 超低功耗

  • 支持FreeRTOS/Zephyr

系统框图

核心架构

  • Real-M300（KM4）

    • Armv8.1-M 架构（兼容 Arm® Cortex®-M55 ISA）

    • 最高频率 345MHz，三级流水线架构

    • 16KB 指令缓存（4路组相联） + 16KB 数据缓存（4路组相联）

    • 集成 FPU/DSP 扩展指令

    • 通过 TrustZone®-M 实现硬件级隔离

  • Real-M200（KM0）

    • Armv8-M 架构（兼容 Arm® Cortex®-M23 ISA）

    • 最高频率 115MHz，两级流水线架构

    • 与 Real-M300 代码兼容，优化至 μA 级功耗

存储系统

  • 片内存储：512KB SRAM

  • 扩展存储（包含KGD）：

    • QSPI NOR Flash（最大频率 104MHz）

    • DQ8 DDR 模式 PSRAM（最大频率 200MHz）

  • 总线架构：多层 AXI 总线并行访问

外设与接口

  • 灵活配置的多功能引脚

  • 多种通信接口：SPI x 2，UART x 4，I2C x 2

  • 硬件Key-Scan接口，最大支持64个按键

  • 硬件IR收发器，易于适配各种IR协议

  • 支持RTC定时器和10路独立定时器

  • 支持8通道PWM定时器和1路捕获定时器

  • 支持7通道通用12位ADC和1个电池量测专用通道

  • 支持4通道触摸按键

  • 支持8独立通道GDMA

  • 支持1路独立看门狗，1路安全区看门狗，2路普通看门狗

  • 支持USB 2.0全速设备模式

  • 支持SDIO设备1位和4位模式

  • 内置串行LED控制器，用于控制外部LED灯

  • 集成像素处理引擎，用于快速处理图像数据

  • 集成OSPI显示接口，支持OSPI、QSPI、SPI接口屏幕

  • 集成Audio Codec，包含2个数字麦克风通道（DMIC）

  • 集成2个I2S接口（最高384kHz采样率，最高支持8通道数据收发）

安全特性

  • 安全启动

  • Arm TrustZone-M 安全架构

  • 真随机数发生器（TRNG）

  • 硬件加密引擎（AES/SHA）

  • Flash全盘/分区加密

  • 读保护机制（RDP）机制

  • JTAG/SWD密码保护

  • 2KB OTP存储器（用户可用2Kb）

  • PSA Level-2 安全认证

无线连接

  • Wi-Fi 4 (802.11a/b/g/n)

    • 1T1R，2.4GHz/5GHz双频

    • 40MHz信道带宽，MCS0 ~ MCS7调制

    • 天线分集技术

    • WoWLAN节能模式

      • 2.4GHz功耗：84μA（DTIM=10，3.3V，25°C）

      • 5GHz功耗：122μA（DTIM=10，3.3V，25°C）

  • Bluetooth 5.0 LE

    • 支持LE Coded PHY长距离模式（125kbps/500kbps）

    • 散射网支持8主设备+3从设备并发

    • 发射功率：-10dBm ~ +10dBm（典型值 8dBm）

    • 接收灵敏度：-99dBm@1Mbps

电源管理

  • 动态电压/频率调节（1.71V ~ 3.63V）

  • 工作模式：

    • Active模式：全速运行（345MHz）

    • Sleep模式：外设和Wi-Fi可以唤醒系统（<100uA）

    • Deep-sleep模式：10.4µA（1.8V，25°C）

封装与可靠性

  • QFN48封装：6mm x 6mm

  • QFN68封装：8mm x 8mm

  • BGA100封装：5.1mm x 5.1mm

  • 工作电压：

    • 常压：2.97V ~ 3.63V

    • 宽压：1.71V ~ 3.63V

  • 工作温度：

    • 常温：-20°C ~ 85°C

    • 宽温：-40°C ~ 105°C

  • ESD防护：HBM ±3.5kV

应用

  • 智能家居

  • 线控器

  • 智能门锁

  • 电池摄像头

  • 智能遥控器

  • Wi-Fi 音箱

  • WHC FullMAC NIC

  • BLE 网关

  • 智能 POS 机

    • TrustZone® 安全域隔离技术

  • Real-M300（KM4，网络处理器）

    • 最大频率333MHz，3级流水线架构

    • 16KB指令缓存（4路组相联） + 32KB数据缓存（4路组相联）

    • Cortex®-M55指令集架构，单精度浮点单元（FPU）/数字信号处理（DSP）/向量扩展指令集（MVE）

    • TrustZone®-M 硬件级隔离

  • Real-M200（KM0，低功耗核）

    • 16KB指令缓存（2路组相联） + 8KB数据缓存（2路组相联）
