Key Features

  • WiFi-4 2.4G/5G Dual-band + Bluetooth 5.0 LE

  • PSA Level-2 Certified

  • Ultra-low Power

  • FreeRTOS/Zephyr supported

Block Diagram

Core Architecture

This dual-core SoC combines high-performance and ultra-low-power processing:

  • Real-M300 (KM4)

    • Armv8.1-M architecture (Cortex®-M55 ISA compatible)

    • 345MHz max frequency , 3-stage pipeline

    • 16KB 4-way I-Cache + 16KB 4-way D-Cache

    • Integrated FPU/DSP extensions , TrustZone®-M isolation

  • Real-M200 (KM0)

    • Armv8-M architecture (Cortex®-M23 ISA compatible)

    • 115MHz max frequency , 2-stage pipeline

    • Code-compatible with KM4 , Optimized for μA-level operation

Memory System

  • On-chip Memory: 512KB SRAM

  • External Memory (including KGD):

    • QSPI NOR Flash (maximum frequency 104MHz)

    • DQ8 DDR mode PSRAM (maximum frequency 200MHz)

  • Bus Architecture: Multi-layer AXI bus with parallel access support

Peripheral Interfaces

  • Flexible design of GPIO configuration

  • Multi-communication interfaces: SPI x 2, UART x 4, I2C x 2

  • Hardware Key-Scan interface supports up to 8*8 (64) keys

  • Hardware IR transceiver can easily adapt to various IR protocols

  • Supports Real-Time Clock timer together with 10 basic timers

  • Supports 8 channels of PWM timer and 1 capture timer

  • Supports 7 channels of general 12-bit ADC and 1 channel of VBAT

  • Supports 4 channels of touch pad

  • Supports 8 independent channels of GDMA

  • Supports USB 2.0 full-speed device mode

  • Supports SDIO device with 1-bit and 4-bit mode

  • Embeds a serial LEDC to control the external LED lamps

  • Integrated Pixel Processing Engine (PPE) to process pixel data faster

  • Integrated OSPI display interface supports screens with OSPI/QSPI/SPI interfaces

  • Integrated audio codec supports 2 channels DMIC interface

  • I2S x 2: up to 384kHz sampling rate

Security Features

  • Secure Boot

  • Arm TrustZone-M security architecture

  • True Random Number Generator (TRNG)

  • Hardware encryption engine (AES/SHA)

  • Flash full-disk/partition encryption

  • Read Protection (RDP) mechanism

  • JTAG/SWD password protection

  • 2KB OTP memory (2K bits available for user)

  • PSA Level 2 Certified

Wireless Connectivity

  • Wi-Fi 4 (802.11a/b/g/n)

    • 802.11 a/b/g/n, 1T1R Dual-band 2.4/5GHz

    • 40MHz channel bandwidth , MCS0-MCS7 modulation

    • Antenna diversity

    • WoWLAN power saving:

      • 84μA@2.4GHz (DTIM=10, 3.3V, 25°C)

      • 122μA@5GHz (DTIM=10, 3.3V, 25°C)

  • Bluetooth 5.0 LE

    • LE Long Range (125/500kbps)

    • Scatternet: 8 masters + 3 slaves

    • TX Power: -10dBm ~ +10dBm (typical 8dBm)

    • RX Sensitivity: -99dBm@1Mbps

Power Management

  • Dynamic Voltage/Frequency Scaling (1.71V-3.63V)

  • Power Modes:

    • Active: Full operation (345MHz)

    • Sleep: Peripheral or Wi-Fi can wakeup (<100uA)

    • Deep Sleep: 10.4µA (1.8V, 25°C)

Package and Reliability

  • QFN48 package, 6mm x 6mm

  • QFN68 package, 8mm x 8mm

  • BGA100 package, 5.1mm x 5.1mm

  • Operating voltage:

    • Standard voltage: 2.97V ~ 3.63V

  • Operating temperature:

    • Standard temperature: -20°C ~ 85°C

    • Wide-range temperature: -40°C ~ 105°C

  • ESD protection: HBM ±3.5kV

    • 400MHz clock frequency with 3-stage pipeline

    • 16KB 4-way instruction cache + 16KB 4-way data cache

    • Hardware-enforced isolation via TrustZone®-M

    • Open-source ISA with custom extension support

Memory System

  • On-chip Memory: 768KB SRAM

  • External Memory (including KGD):

    • QSPI NOR Flash (maximum frequency 104MHz)

  • Bus Architecture: Multi-layer AXI bus with parallel access support
